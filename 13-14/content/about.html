О приложении

Заметки PWA — прогрессивное веб-приложение для управления заметками.

Возможности:

  • ✅ Поиск и фильтрация заметок
  • ✅ Выбор цвета для каждой заметки
  • ✅ Редактирование и удаление
  • ✅ Отметка «выполнено»
  • ✅ Работа без интернета (офлайн-режим)
  • ✅ Мгновенная загрузка (App Shell)
  • ✅ Синхронизация в реальном времени (WebSocket)
  • ✅ Push-уведомления о новых заметках

Технологии:

Service Worker • Web App Manifest • localStorage • Socket.io • Push API
